\doxysubsection{Detailed Description}
Mutex operation tracker. 

Records precise timing of mutex state transitions with nanosecond resolution for contention analysis.

Stores data as \mbox{[}request\+\_\+time, acquire\+\_\+time, release\+\_\+time\mbox{]} tuples indexed by mutex name.
