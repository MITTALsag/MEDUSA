\doxysubsection{Detailed Description}
Fundamental executable job unit. 

Encapsulates work items with\+:
\begin{DoxyItemize}
\item Unique identification and hierarchical relationships
\item Precise timing instrumentation
\item Execution status tracking
\item Callback function management
\end{DoxyItemize}

Forms the atomic unit of work in the dispatch queue system.
